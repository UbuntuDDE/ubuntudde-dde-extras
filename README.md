# ubuntudde-dde-extras
**Deepin Desktop Environment Extras for Ubuntu DDE**

This is a meta package depends DDE Extras.

The package installs the following packages:

* dde-calendar
* deepin-calculator
* deepin-boot-maker
* deepin-clone
* deepin-albumn
* deepin-image-viewer
* deepin-movie-reborn
* deepin-music
* deepin-editor
* deepin-screen-recorder
* deepin-screenshot
* deepin-system-monitor
* deepin-terminal
* deepin-turbo
